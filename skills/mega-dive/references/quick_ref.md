# Mega Dive - Quick Reference

## Triggers
- "deep dive"
- "mega dive"  
- "research everything"
- "full research"

## 3-Phase Process

### Phase 1: Analyze (ReAct)
1. THOUGHT - Define scope
2. ACT - Research
3. OBSERVATION - Incorporate
4. RESPOND - Present with confidence
5. SOURCE CITATION - Always cite!
6. CONFIDENCE - Use levels
7. SELF-VERIFICATION - Check facts
8. MULTI-SOURCE - Verify claims

### Phase 2: Brainstorm
Use `skills/brainstorm/SKILL.md`
- DIVERGENCE → CROSS-POLLINATION → AMPLIFICATION → PRESENT
- GOAL SETTING → RISK → SCORING → SAVE → ACTIONS → TAGS

### Phase 3: Upgrades
1. ANALYZE & RESEARCH
2. SEARCH WEB
3. RESEARCH ISSUES
4. CHECK CURRENT SETUP
5. CHECK HINDSIGHT
6. DETAIL EACH UPGRADE
7. COMPARE ALTERNATIVES
8. PRIORITIZE
9. VERSION TRACKING
10. AUTO-BACKUP
11. ROI/COST
12. DEPENDENCY CHECK

## Output
One unified report with all 3 phases

## Time
25-50 minutes
