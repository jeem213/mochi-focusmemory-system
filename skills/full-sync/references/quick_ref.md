# Full Sync - Quick Reference

## Triggers
- "full sync"
- "sync everything"
- "complete sync"

## 13 Steps
1. HINDSIGHT HEALTH CHECK (18-point via skills/hindsight)
2. DIRECTIVES - Query active rules
3. REFLECT - Synthesize memories (90s)
4. INGEST - Read GitHub state
5. RECALL - Search design discussions
6. CROSS-REFERENCE - Memory vs code
7. CRON STATUS - Check HEARTBEAT.md
8. CHANGE DETECTION - New commits/memories
9. SYNC TIMESTAMPS - Track last sync
10. HEALTH SCORING - Rate 1-10
11. SYNC VALIDATION - Test integrations
12. CONFLICT DETECTION - Find contradictions
13. OUTPUT COMPLETE REPORT

## Output
- Health score (1-10)
- System status table
- Active directives
- Changes detected
- Cron jobs status
- Conflicts (if any)
